Aphasia Task Suite

View on GitHub →

A comprehensive suite of 32 web-based diagnostic tasks designed for aphasia research. Each task is deployed via Qualtrics and features millisecond-precision timing for synchronization with EEG recording equipment.

The tasks assess various language functions including comprehension, production, repetition, and naming — providing researchers with granular data on language processing in patients with aphasia.

JavaScript HTML/CSS Qualtrics Research

EEG VR Stimuli Engine

View on GitHub →

VR applications for EEG data collection built for Oculus Quest 2. The engine provides precise audiovisual synchronization critical for neuroscience experiments that correlate brain activity with sensory stimuli.

Features include customizable experiment protocols, real-time stimulus presentation, and integration with external EEG recording systems via network triggers.

C# Unity Meta Quest SDK VR/AR

Autograder

View on GitHub →

An automated unit testing system for C++ programming assignments. The autograder runs student submissions against a comprehensive test suite, providing detailed feedback and grades to hundreds of students each semester.

Built to streamline the grading workflow for Georgetown's CS department, ensuring consistent and fair evaluation across all submissions.

C++ Bash Python Education

Codon Reader

View on GitHub →

A file-based analysis tool for DNA nucleotide and protein codon sequencing. The software reads genetic sequence files and provides detailed analysis of codon usage, amino acid translation, and sequence statistics.

Designed for bioinformatics workflows, supporting common sequence file formats and providing exportable analysis reports.

C++ Bioinformatics File I/O
